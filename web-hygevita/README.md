# Hygevita Web

Sitio web público de **Hygevita** — plataforma de gestión integral para clínicas y farmacias.

Hygevita es una aplicación de **MedFarmaSoft**.

## Stack
- React 19 + Vite
- React Router
- Tailwind CSS
- Lucide icons
- Cloudflare Turnstile

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
